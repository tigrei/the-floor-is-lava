MECHANICAL ISSUES DATA DICTIONARY

Dataset Purpose
  This dataset defines potential mechanical failures that may occur during a naval transit simulation. Each record represents a single mechanical issue that can affect vessel performance, mission readiness, fuel consumption, speed, and operational capability.

Field Definitions

  id
    Type: String
    Description:
      Unique identifier assigned to each mechanical issue. The identifier should remain constant across all simulation runs and datasets.
    Example:
      ENG_001
  
  name
    Type: String
    Description:
      Human-readable title of the mechanical issue. This field provides a concise description of the failure event.
    Example:
      Main Engine Overheating

  category
    Type: String (Categorical)
    Description:
      Primary ship system affected by the issue.
    Allowed Values:
      * Propulsion
      * Electrical
      * Navigation
      * Communications
      * Auxiliary
      * Hull

  severity
    Type: String (Categorical)
    Description:
      Represents the operational seriousness of the issue and its impact on mission execution.
    Allowed Values:
      * Low
      * Medium
      * High
      * Critical

  probability_weight
    Type: Integer
    Description:
      Relative occurrence weight used during random event generation. Larger values indicate a higher likelihood of occurrence compared to other issues.
    Example:
      15
  
  repair_time_hours
    Type: Integer
    Units:
      Hours
    Description:
      Estimated time required to repair the issue and restore affected functionality.
    Example:
      12

  fuel_penalty_percent
    Type: Float
    Units:
      Percent (%)
    Description:
      Additional fuel consumption caused by the issue. A value of 10 indicates fuel consumption increases by 10 percent while the issue remains active.
    Example:
      10.0
  
  speed_penalty_percent
    Type: Float
    Units:
      Percent (%)
    Description:
      Reduction in maximum vessel speed resulting from the issue.
    Example:
      25.0
  
  readiness_penalty
    Type: Integer
  Units:
    Readiness Points
  Description:
    Reduction applied to the vessel readiness score when the issue occurs.
  Example:
    20

  description
    Type: String
    Description:
      Detailed explanation of the issue, its cause, and operational consequences.
    Example:
      Cooling system performance degrades, causing engine temperatures to rise.
